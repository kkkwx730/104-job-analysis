# 台灣旅遊景點分析系統

本專題使用 Python、Flask、Playwright、PostgreSQL 與 Render 建立台灣旅遊景點資料分析網站。

## 使用技術

- Python
- Flask
- Playwright
- PostgreSQL
- Pandas
- Matplotlib
- Bootstrap
- Render
- GitHub

## 系統功能

- 景點資料爬取
- 景點資料儲存
- 景點資料查詢
- 關鍵字搜尋
- 各縣市景點數量分析
- 景點類型分析
- 長條圖與圓餅圖視覺化

## 資料庫

本專題使用 Render PostgreSQL 作為雲端資料庫，儲存爬取後的台灣景點資料。
